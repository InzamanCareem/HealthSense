 # 🏥 HealthSense: Disease Outbreak Analysis System

**HealthSense** is a console based analytics system developed to help 
epidemiologists monitor, sort, search, and analyze outbreak data from multiple 
hospitals and regions. Your system must process time-series records of patient 
counts, identify hotspots, and allow queries for diagnosis frequency, trend
sorting, and forecasting based on structured data.

---

## 📌 Features

- ✅ Add region-based weekly reports from multiple hospitals
- 🔍 Search hospitals by disease name or hospital name and patient count
- 📊 Sort diseases by count or a disease by week
- 🧠 Analyze disease outbreaks by severity using a Binary Search Tree
- 🔁 Undo latest operation using a stack
- 📋 View and manage outbreak reports in a queue
- 🖥️ Command line interface
